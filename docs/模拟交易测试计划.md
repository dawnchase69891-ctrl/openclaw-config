## 🎯 测试目标

### 核心目标

```
1. 验证选股策略有效性
2. 优化交易纪律执行
3. 找到最优胜率策略
4. 找到最高收益策略
5. 建立可复制的交易体系
```

### 量化指标

| 指标 | 基线 | 目标 | 挑战 |
|------|------|------|------|
| **胜率** | 50% | 60% | 70% |
| **平均收益** | 5% | 10% | 15% |
| **最大回撤** | <20% | <15% | <10% |
| **盈亏比** | 2:1 | 3:1 | 4:1 |
| **夏普比率** | >1 | >1.5 | >2 |

## 🔧 策略调整机制

### 调整触发条件

```
触发调整:
- 连续 3 日亏损 → 检查策略
- 连续 5 日亏损 → 暂停交易 + 复盘
- 单周亏损>10% → 降低仓位 + 调整
- 胜率<40% → 策略参数优化
- 回撤>15% → 空仓观望

调整流程:
1. ClawGuard 发出预警
2. financial-agent-core 分析原因
3. backtest-expert 回测验证
4. 骐骥审批调整方案
5. 执行调整
```

### 参数优化范围

| 策略 | 可调参数 | 调整范围 |
|------|---------|---------|
| S01 龙头首阴 | 连板数、止损位 | 2-5 板、5%-10% |
| S02 突破新高 | 突破幅度、成交量 | 3%-10%、1.5-3 倍 |
| S03 回踩均线 | 均线类型、缩量比例 | 10/20/60 日、30%-70% |
| S04 机构共振 | 机构数量、净买入 | 1-3 家、3000 万 -1 亿 |
