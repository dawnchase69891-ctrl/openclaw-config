## 一、前置要求

### 1.1 系统要求

- **操作系统**: Linux / macOS / Windows (WSL)
- **Python**: 3.11+
- **OpenClaw**: 已安装并配置

### 1.2 依赖检查

```bash
# 检查 Python 版本
python3 --version  # 应该 >= 3.11

# 检查 OpenClaw
```

### 1.3 权限要求

- 飞书应用 API 权限（如需飞书集成）
- GitHub 仓库写入权限（如需 GitHub Actions）

## 二、安装步骤

### 2.1 克隆/更新代码

```bash
cd ~/.openclaw/workspace

# 如果是 Git 仓库
git pull origin main

# 或手动下载文件
```

### 2.2 安装 Python 依赖

```bash
cd ~/.openclaw/workspace

# 创建虚拟环境（推荐）
python3 -m venv .venv
source .venv/bin/activate  # Linux/macOS
# .venv\Scripts\activate  # Windows

# 安装依赖
pip install pytest pytest-mock pytest-asyncio
```

### 2.3 创建必要目录

```bash
# 创建模板目录
mkdir -p ~/.openclaw/workspace/docs/templates

# 创建测试目录

# 创建死信队列目录
mkdir -p ~/.openclaw/workspace/.dead-letter

# 创建备份目录
mkdir -p ~/.openclaw/workspace/.backup/config
```

### 2.4 验证文件结构

## 三、配置说明

### 3.1 飞书 API 配置（可选）

如需飞书文档自动创建，配置以下环境变量：

```bash
# ~/.openclaw/workspace/.env
FEISHU_APP_ID=cli_xxxxxxxxxxxxx
FEISHU_APP_SECRET=xxxxxxxxxxxxxxxxxxxxx
FEISHU_BITABLE_TOKEN=bascnxxxxxxxxxxxx
```

### 3.2 文件夹 Token 配置

编辑 `scripts/feishu_doc_creator.py` 中的 `TEMPLATES` 配置：

```python
TEMPLATES = {
    'task-template': DocTemplate(
        name='任务文档模板',
        file='task-template.md',
        description='用于创建任务追踪文档',
        default_folder='UDARfZ30YlI7ild68FmcEoianSg'  # 修改为你的文件夹 token
    ),
    # ...
}
```

### 3.3 任务队列配置

编辑 `scripts/task_queue_manager.py` 中的默认配置：

```python
manager = TaskQueueManager(
    max_concurrent=5,  # 根据系统性能调整
    state_file='/path/to/state.json'  # 可选
)
```

### 3.4 重试策略配置

编辑 `scripts/collaboration_utils.py` 中的重试配置：

```python
self.max_retries = 3      # 最大重试次数
self.base_delay = 5.0     # 基础延迟（秒）
self.max_delay = 60.0     # 最大延迟（秒）
self.jitter = 0.1         # 随机抖动比例
```

## 五、常见问题

### 5.1 安装问题

#### 问题 1: Python 版本过低

```bash
# 错误信息:
# SyntaxError: invalid syntax

# 解决方案:
python3 --version  # 检查版本
# 如果 < 3.11，升级 Python
```

#### 问题 2: 依赖缺失

```bash
# 错误信息:
# ModuleNotFoundError: No module named 'pytest'

# 解决方案:
pip install pytest pytest-mock pytest-asyncio
```

### 5.2 配置问题

#### 问题 1: 模板文件不存在

```bash
# 错误信息:
# FileNotFoundError: [Errno 2] No such file or directory: 'task-template.md'

# 解决方案:
# 1. 检查模板目录
ls -la ~/.openclaw/workspace/docs/templates/

# 2. 如果文件不存在，重新创建
# 参考 docs/templates/ 目录下的模板文件
```

#### 问题 2: 文件夹 Token 错误

```bash
# 错误信息:
# 飞书 API 返回：folder not found

# 解决方案:
# 1. 检查 folder_token 是否正确
# 2. 确保你有该文件夹的写入权限
# 3. 从飞书 URL 获取正确的 token
```

### 5.3 运行问题

#### 问题 1: Agent 离线

```bash
# 错误信息:
# AgentOfflineError: Agent clawbuilder 离线

# 解决方案:
# 1. 检查 Agent 是否启动
openclaw sessions list

# 2. 检查 OpenClaw 网关状态
openclaw gateway status

# 3. 如果网关未启动，启动它
openclaw gateway start
```

#### 问题 2: 任务超时

```bash
# 错误信息:
# asyncio.TimeoutError

# 解决方案:
# 1. 增加 timeout 参数
await manager.spawn_with_control(
    agent_id='clawbuilder',
    task='复杂任务',
    timeout=600  # 增加到 10 分钟
)

# 2. 检查任务是否过于复杂
# 3. 检查系统资源是否充足
```

#### 问题 3: 测试失败

```bash
# 错误信息:
# FAILED tests/test_integration.py::TestEndToEnd::test_xxx

# 解决方案:
# 1. 查看详细错误信息
pytest tests/test_integration.py::TestEndToEnd::test_xxx -v -s

# 2. 检查测试依赖
# 3. 确保测试环境配置正确
```

### 5.4 飞书集成问题

#### 问题 1: API 认证失败

```bash
# 错误信息:
# Invalid app_id or app_secret

# 解决方案:
# 1. 检查 .env 文件中的凭证
# 2. 确保飞书应用已启用
# 3. 检查应用权限配置
```

#### 问题 2: 文档创建失败

```bash
# 错误信息:
# 飞书 API 返回：permission denied

# 解决方案:
# 1. 检查应用是否有文档创建权限
# 2. 检查文件夹权限
# 3. 确保文件夹 token 正确
```
