## 二、交易纪律标准 ⭐⭐⭐⭐⭐

### 2.1 买入纪律

#### 买入前检查清单

```
买入前必须确认 (全部打勾方可执行):

□ 1. 综合评分 > 80 分
□ 2. 所属板块为热点
□ 3. 主力净流入 > 0
□ 4. 无重大利空消息
□ 5. 已制定止损位
□ 6. 仓位计算完成
□ 7. 骐骥审批通过

签字确认：________  日期：________
```

#### 买入时机标准

| 买点类型 | 条件 | 仓位 | 优先级 |
|---------|------|------|-------|
| **龙头首阴** | 连板≥3+ 首阴 + 非跌停 | 20% | ⭐⭐⭐⭐⭐ |
| **突破新高** | 突破平台 + 放量 | 20% | ⭐⭐⭐⭐⭐ |
| **回踩均线** | 回踩 10 日线 + 缩量 | 15% | ⭐⭐⭐⭐ |
| **早盘强势** | 高开 3%-5%+ 放量 | 10% | ⭐⭐⭐ |

#### 禁止买入情形

```
❌ 以下情况禁止买入:

1. 大盘跌破 20 日均线
2. 个股跌破 20 日均线
3. 有重大利空消息
4. 财报披露前 3 日
5. 连续大涨后 (>50%)
6. 成交量异常放大 (3 倍以上)
7. 未制定止损计划
8. 综合评分 < 80 分

违反禁令：记过一次，罚款____元
```

---

### 2.2 卖出纪律

#### 止盈标准

| 涨幅 | 操作 | 说明 |
|------|------|------|
| **20%-30%** | 减仓 1/3 | 锁定部分利润 |
| **30%-50%** | 再减仓 1/3 | 继续锁定利润 |
| **>50%** | 清仓或留底仓 | 防止坐过山车 |

#### 止损标准

| 策略类型 | 止损位 | 执行要求 |
|---------|-------|---------|
| **短线** | -8% | 无条件执行 |
| **中线** | -15% | 严格执行 |
| **长线** | -20% 或基本面恶化 | 基本面优先 |
| **技术止损** | 关键支撑位 | 如 20 日/60 日线 |

#### 强制卖出情形

```
❗ 以下情况必须卖出:

1. 触及止损位 → 立即卖出
2. 基本面恶化 → 立即卖出
3. 重大利空 → 立即卖出
4. 跌破重要支撑 → 当日卖出
5. 连续 3 日不涨 → 考虑卖出
6. 发现更好标的 → 换股
7. 仓位超限 → 减仓至标准
8. 大盘破位 → 降低仓位

违反纪律：记过 + 罚款 + 复盘检讨
```

---

### 2.3 仓位管理标准

#### 总仓位控制

| 大盘位置 | 总仓位 | 说明 |
|---------|-------|------|
| **>60 日线** | 80%-100% | 进攻阶段 |
| **20 日线 -60 日线** | 50%-80% | 防守阶段 |
| **<20 日线** | 0%-30% | 空仓观望 |

#### 单只股票仓位

| 风险等级 | 单只上限 | 说明 |
|---------|---------|------|
| **低风险** (长线价值) | 40% | 行业龙头 |
| **中风险** (中线成长) | 30% | 业绩增长 |
| **高风险** (短线热点) | 20% | 题材炒作 |

#### 仓位计算公式

```python
单只股票仓位 = min(
    总仓位 × 20%,  # 不超过总仓位 20%
    可用资金 × 50%,  # 不超过可用资金 50%
    风险等级上限    # 按风险等级确定
)
```

### 2.4 风险控制标准

#### 风险等级评估

| 指标 | 低风险 | 中风险 | 高风险 |
|------|-------|-------|-------|
| **大盘位置** | >60 日线 | 20-60 日线 | <20 日线 |
| **持仓数量** | 3-5 只 | 5-8 只 | >8 只 |
| **单只仓位** | <20% | 20%-30% | >30% |
| **止损执行** | 严格 | 一般 | 松散 |

#### 风险预警机制

| 预警级别 | 触发条件 | 应对措施 |
|---------|---------|---------|
| **蓝色预警** | 单只亏损>5% | 关注，准备止损 |
| **黄色预警** | 单只亏损>8% | 减仓，严格执行止损 |
| **橙色预警** | 总亏损>10% | 降低总仓位至 50% |
| **红色预警** | 总亏损>20% | 空仓观望，全面复盘 |

## 三、每日工作流程 ⭐⭐⭐⭐⭐

### 3.1 盘前准备 (9:00-9:25)

```
时间：9:00-9:25
负责人：a-stock-monitor + ClawSupport

工作内容:
□ 1. 查看隔夜外盘走势
□ 2. 查看财经新闻
□ 3. 查看龙虎榜数据 (如有)
□ 4. 查看北向资金流向
□ 5. 更新股票池
□ 6. 准备集合竞价观察

输出:
- 盘前简报 (飞书推送)
- 今日关注股票列表
```

---

### 3.2 盘中监控 (9:30-11:30, 13:00-15:00)

```
时间：盘中实时
负责人：a-stock-monitor (自动) + ClawGuard

工作内容:
□ 1. 监控资金流向
□ 2. 监控龙虎榜 (14:30 开始)
□ 3. 监控 RPS 排名
□ 4. 监控买卖点触发
□ 5. 执行交易纪律检查

输出:
- 实时告警 (飞书)
- 异常交易提醒
- 纪律执行记录
```

---

### 3.3 盘后总结 (15:00-17:00)

```
时间：15:00-17:00
负责人：ClawCoordinator + financial-agent-core

工作内容:
□ 1. 统计今日交易
□ 2. 更新持仓盈亏
□ 3. 分析龙虎榜数据
□ 4. 复盘今日操作
□ 5. 制定明日计划

输出:
- 交易日报 (飞书)
- 龙虎榜分析
- 明日选股推荐
```

---

### 3.4 每周复盘 (周五 17:00-18:00)

```
时间：每周五 17:00-18:00
负责人：ClawCoordinator + 骐骥

工作内容:
□ 1. 统计本周交易
□ 2. 分析盈亏原因
□ 3. 检查纪律执行
□ 4. 评估选股策略
□ 5. 制定下周计划

输出:
- 周度交易报告
- 纪律执行评分
- 策略优化建议
- 下周操作计划
```

## 五、违规处理 ⭐⭐⭐⭐⭐

| 等级 | 违规行为 | 处理措施 |
|------|---------|---------|
| **一级违规** | 未填写检查清单 | 警告 + 补填 |

| **五级违规** | 明知故犯/隐瞒 | 暂停交易 + 罚款 1000 元 + 降职 |
