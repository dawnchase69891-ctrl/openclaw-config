## 一、龙虎榜选股法 ⭐⭐⭐⭐⭐

### 1.1 龙虎榜数据解读

#### 核心指标

| 指标 | 含义 | 关注值 |
|------|------|-------|
| **买入金额** | 主力买入力度 | >5000 万 |
| **买入占比** | 买入占成交比例 | >10% |
| **机构席位** | 机构参与度 | ≥2 家 |
| **游资席位** | 游资活跃度 | 知名游资 |
| **净买入** | 主力净流向 | >3000 万 |
| **营业部排名** | 席位实力 | 前 5 知名 |

#### 龙虎榜类型

| 类型 | 条件 | 参考价值 |
|------|------|---------|
| **日涨幅偏离值≥7%** | 强势股 | ⭐⭐⭐⭐⭐ |
| **日换手率≥20%** | 高活跃 | ⭐⭐⭐⭐ |
| **日价格振幅≥15%** | 大幅波动 | ⭐⭐⭐ |
| **连续 3 日涨幅偏离≥20%** | 持续强势 | ⭐⭐⭐⭐⭐ |

### 1.2 龙虎榜选股策略

#### 策略 1：机构席位追踪

```python
# 选股条件
条件:
  - 机构席位买入 ≥ 2 家
  - 机构净买入 > 3000 万
  - 机构买入占比 > 10%
  - 无机构大额卖出

逻辑:
  - 机构调研深入，信息优势
  - 资金体量大，持续性强
  - 多为中线布局，稳定性高

案例:
  2026-03-XX: XXXX 机构大举买入，后续 5 日涨幅 35%
```

#### 策略 2：知名游资追踪

```python
# 知名游资席位
游资席位:
  - 中信证券上海分公司
  - 国泰君安上海江苏路
  - 华泰证券深圳益田路
  - 中国银河绍兴
  - 招商证券深圳深南东路

选股条件:
  - 知名游资买入
  - 游资买入 > 2000 万
  - 游资买入占比 > 5%
  - 配合题材热点

逻辑:
  - 游资嗅觉灵敏，善于捕捉热点
  - 资金实力强，拉升意愿足
  - 但注意游资快进快出特性
```

#### 策略 3：机构 + 游资共振

```python
# 最强形态
条件:
  - 机构席位买入 ≥ 1 家
  - 知名游资买入 ≥ 1 家
  - 合计净买入 > 5000 万
  - 买入占比 > 15%

逻辑:
  - 机构提供基本面背书
  - 游资提供市场热度
  - 合力最强，持续性最好

信号强度: ⭐⭐⭐⭐⭐
```

### 1.3 龙虎榜陷阱识别

#### 陷阱 1：对倒出货

```
特征:
  - 同一营业部既买又卖
  - 买入金额大但净买入少
  - 股价高位放量

识别:
  - 查看买卖前五是否有关联席位
  - 计算净买入而非总买入
  - 结合股价位置判断
```

#### 陷阱 2：一日游行情

```
特征:
  - 纯游资炒作，无机构参与
  - 题材炒作末期
  - 成交量异常放大

识别:
  - 查看是否有机构参与
  - 分析题材持续性
  - 观察量能是否健康
```

#### 陷阱 3：接盘侠

```
特征:
  - 买方金额大但分散
  - 卖方集中且金额大
  - 股价高位滞涨

识别:
  - 对比买卖双方集中度
  - 分析股价位置
  - 观察后续走势
```

---

### 2.2 主力资金流入选股

#### 核心指标

```python
主力流入指标:
  - 主力净流入 > 5000 万
  - 主力流入占比 > 10%
  - 连续 3 日净流入
  - 大单净流入 > 中单 + 小单

量价配合:
  - 放量上涨 (健康)
  - 缩量回调 (正常)
  - 放量下跌 (警惕)
  - 缩量上涨 (观察)
```

#### 选股公式

```python
# 强势股资金流入选股
条件:
  1. 主力净流入 > 5000 万
  2. 主力流入占比 > 10%
  3. 股价涨幅 3%-7% (非涨停)
  4. 成交量 > 5 日均量 1.5 倍
  5. 股价在 20 日均线上方
  6. 所属板块为热点

信号强度: ⭐⭐⭐⭐⭐
```

## 三、龙头股捕捉策略 ⭐⭐⭐⭐⭐

### 3.1 龙头股特征

#### 基本面特征

| 特征 | 描述 | 权重 |
|------|------|------|
| **行业地位** | 行业前三，市占率高 | ⭐⭐⭐⭐⭐ |
| **业绩增长** | 连续 3 年增长>20% | ⭐⭐⭐⭐⭐ |
| **盈利能力** | ROE>15%, 毛利率>30% | ⭐⭐⭐⭐ |
| **研发投入** | 研发占比>5% | ⭐⭐⭐⭐ |
| **护城河** | 品牌/技术/规模优势 | ⭐⭐⭐⭐⭐ |

#### 技术面特征

| 特征 | 描述 | 权重 |
|------|------|------|
| **趋势** | 均线多头排列 | ⭐⭐⭐⭐⭐ |
| **量能** | 持续放量上涨 | ⭐⭐⭐⭐⭐ |
| **形态** | 突破平台/新高 | ⭐⭐⭐⭐ |
| **强度** | 相对强度 RPS>90 | ⭐⭐⭐⭐⭐ |
| **资金** | 主力持续流入 | ⭐⭐⭐⭐⭐ |

---

### 3.2 龙头股识别方法

#### 方法 1：板块强度排序

```python
# 步骤
1. 确定热点板块 (政策/业绩/题材)
2. 计算板块内个股 RPS(相对强度)
3. RPS 排名前 3 为候选龙头
4. 结合资金流入确认

RPS 计算:
  RPS = 个股涨幅排名 / 总股票数 * 100
  RPS > 90: 强势股
  RPS > 95: 龙头候选
```

#### 方法 2：连板高度识别

```python
# 连板龙头特征
条件:
  - 连板数 ≥ 3
  - 封板质量好 (早盘封板)
  - 成交量健康 (非巨量)
  - 板块跟风股多

逻辑:
  - 连板高度代表市场认可度
  - 封板质量代表主力决心
  - 板块效应代表持续性
```

#### 方法 3：资金认可度

```python
# 资金认可指标
指标:
  - 龙虎榜机构买入 ≥ 2 家
  - 主力净流入连续 5 日 > 0
  - 北向资金持续加仓
  - 融资余额持续增加

逻辑:
  - 多路资金认可
  - 资金持续流入
  - 行情持续性强
```

---

### 3.3 龙头股买入时机

#### 买点 1：首次分歧

```python
# 龙头首阴战法
条件:
  - 龙头股连续涨停 ≥ 3
  - 首次收阴线 (非跌停)
  - 成交量放大但非巨量
  - 板块热度仍在

买入:
  - 次日低开高走时
  - 或盘中翻红时

止损:
  - 跌破首阴最低价
  - 或亏损>8%

信号强度: ⭐⭐⭐⭐⭐
```

#### 买点 2：回踩均线

```python
# 龙头回踩 10 日线
条件:
  - 龙头股上升趋势明确
  - 回踩 10 日均线
  - 缩量回调
  - 均线支撑有效

买入:
  - 回踩 10 日线企稳时
  - 或重新放量上涨时

止损:
  - 跌破 20 日均线
  - 或亏损>10%

信号强度: ⭐⭐⭐⭐
```

#### 买点 3：突破新高

```python
# 龙头突破新高
条件:
  - 股价创阶段新高
  - 成交量放大
  - 板块配合
  - 无重大利空

买入:
  - 突破瞬间
  - 或回踩确认时

止损:
  - 跌回突破平台下方
  - 或亏损>8%

信号强度: ⭐⭐⭐⭐⭐
```

## 四、综合选股策略 ⭐⭐⭐⭐⭐

### 4.1 三位一体选股法

```python
# 基本面 + 技术面 + 资金面

基本面 (40% 权重):
  - 行业景气度高
  - 业绩增长>20%
  - ROE>15%
  - 行业地位前三

技术面 (30% 权重):
  - 均线多头排列
  - RPS>90
  - 突破平台或新高
  - 量价配合健康

资金面 (30% 权重):
  - 主力净流入>5000 万
  - 龙虎榜机构买入
  - 北向资金加仓
  - 融资余额增加

综合评分>80 分：强烈关注
综合评分>70 分：关注
综合评分>60 分：观察
```

---

### 4.2 选股流程

```
1. 确定热点板块
   └─→ 政策导向/业绩驱动/题材炒作

2. 板块内筛选
   └─→ RPS 排序 → 前 10 名

3. 基本面过滤
   └─→ 业绩增长/ROE/行业地位

4. 资金面确认
   └─→ 主力流入/龙虎榜/北向资金

5. 技术面择时
   └─→ 买点信号/止损位

6. 仓位管理
   └─→ 分批建仓/动态调整
```

## 五、交易策略 ⭐⭐⭐⭐⭐

### 5.1 短线策略 (1-5 日)

```python
# 龙头股短线战法

选股:
  - 连板数 ≥ 2

卖出:
  - 涨停持有

  - 或跌破 5 日线

止损:
  - 亏损>8% 止损
  - 或跌破重要支撑

仓位:
  - 单只股票 ≤ 20%
```

### 5.2 中线策略 (1-4 周)

```python
# 机构票中线战法

选股:

买入:
  - 突破后回踩确认

  - 涨幅>30% 分批减仓
  - 或跌破 60 日线

止损:

仓位:
```

---

### 5.3 长线策略 (1-12 月)

```python
# 价值投资长线战法

选股:

  - 业绩稳定增长
  - 护城河深

买入:
  - 估值合理 (PE<30)
  - 或市场大跌时

卖出:
  - 估值过高 (PE>50)

止损:
  - 基本面恶化立即卖出
  - 技术面止损放宽至 20%

仓位:
  - 单只股票 ≤ 40%
```

## 六、风险控制 ⭐⭐⭐⭐⭐

### 6.1 仓位管理

```python
# 金字塔仓位管理

行情好 (大盘>20 日线):
  - 总仓位：70%-100%
  - 单只：≤30%

行情一般 (大盘震荡):
  - 总仓位：40%-70%
  - 单只：≤20%

行情差 (大盘<20 日线):
  - 总仓位：0%-40%
  - 单只：≤10%
```

---

### 6.2 止损策略

| 策略类型 | 止损位 | 说明 |
|---------|-------|------|
| **短线** | -8% | 严格执行 |
| **中线** | -15% | 可适度放宽 |
| **长线** | -20% 或基本面恶化 | 基本面优先 |
| **技术止损** | 关键支撑位 | 如 20 日/60 日线 |

---

### 6.3 止盈策略

```python
# 分批止盈

涨幅 20%-30%: 减仓 1/3
涨幅 30%-50%: 再减仓 1/3
涨幅>50%: 清仓或留底仓

逻辑:
  - 让利润奔跑
  - 但锁定部分利润
  - 避免坐过山车
```
