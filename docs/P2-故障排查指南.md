## 一、快速诊断流程

### 1.1 诊断检查清单

按顺序检查以下项目：

```bash
# 1. 检查 Python 环境
python3 --version  # 应该 >= 3.11

# 2. 检查依赖
pip list | grep pytest  # 应该显示 pytest

# 3. 检查文件结构
ls -la ~/.openclaw/workspace/scripts/
ls -la ~/.openclaw/workspace/docs/templates/
ls -la ~/.openclaw/workspace/tests/

# 4. 检查 OpenClaw 状态
openclaw gateway status
openclaw sessions list

# 5. 运行快速测试
cd ~/.openclaw/workspace
python3 scripts/feishu_doc_creator.py list-templates
```

### 1.2 诊断脚本

创建诊断脚本 `scripts/diagnose.py`:

```python
#!/usr/bin/env python3
# -*- coding: utf-8 -*-
"""
P2 模块诊断脚本
"""

import sys
import subprocess
from pathlib import Path

WORKSPACE = Path.home() / '.openclaw' / 'workspace'

def check_python_version():
    """检查 Python 版本"""
    version = sys.version_info
    print(f"Python 版本：{version.major}.{version.minor}.{version.micro}")
    return version.major >= 3 and version.minor >= 11

def check_dependencies():
    """检查依赖"""
    deps = ['pytest', 'pytest-mock', 'pytest-asyncio']
    missing = []
    
    for dep in deps:
        try:
            __import__(dep.replace('-', '_'))
        except ImportError:
            missing.append(dep)
    
    if missing:
        print(f"❌ 缺少依赖：{', '.join(missing)}")
        print(f"   运行：pip install {' '.join(missing)}")
        return False
    
    print("✅ 依赖检查通过")
    return True

def check_files():
    """检查文件结构"""
    required_files = [
        WORKSPACE / 'scripts' / 'feishu_doc_creator.py',
        WORKSPACE / 'scripts' / 'collaboration_utils.py',
        WORKSPACE / 'scripts' / 'task_queue_manager.py',
        WORKSPACE / 'docs' / 'templates' / 'task-template.md',
        WORKSPACE / 'docs' / 'templates' / 'decision-template.md',
        WORKSPACE / 'docs' / 'templates' / 'meeting-template.md',
        WORKSPACE / 'docs' / 'templates' / 'daily-report-template.md',
        WORKSPACE / 'tests' / 'conftest.py',
        WORKSPACE / 'tests' / 'test_integration.py',
    ]
    
    missing = [f for f in required_files if not f.exists()]
    
    if missing:
        print(f"❌ 缺少文件:")
        for f in missing:
            print(f"   - {f}")
        return False
    
    print("✅ 文件结构检查通过")
    return True

def check_openclaw():
    """检查 OpenClaw 状态"""
    try:
        result = subprocess.run(
            ['openclaw', 'gateway', 'status'],
            capture_output=True,
            text=True,
            timeout=10
        )
        
        if result.returncode == 0:
            print("✅ OpenClaw 网关运行正常")
            return True
        else:
            print(f"⚠️ OpenClaw 网关状态异常：{result.stderr}")
            return False
            
    except Exception as e:
        print(f"❌ 无法检查 OpenClaw: {e}")
        return False

def run_quick_test():
    """运行快速测试"""
    try:
        result = subprocess.run(
            ['python3', 'scripts/feishu_doc_creator.py', 'list-templates'],
            capture_output=True,
            text=True,
            timeout=10,
            cwd=WORKSPACE
        )
        
        if result.returncode == 0:
            print("✅ 快速测试通过")
            return True
        else:
            print(f"❌ 快速测试失败：{result.stderr}")
            return False
            
    except Exception as e:
        print(f"❌ 快速测试异常：{e}")
        return False

def main():
    """主诊断流程"""
    print("=" * 60)
    print("P2 模块诊断工具")
    print("=" * 60)
    print()
    
    checks = [
        ("Python 版本", check_python_version),
        ("依赖检查", check_dependencies),
        ("文件结构", check_files),
        ("OpenClaw 状态", check_openclaw),
        ("快速测试", run_quick_test),
    ]
    
    results = []
    
    for name, check in checks:
        print(f"\n检查：{name}")
        print("-" * 40)
        result = check()
        results.append((name, result))
    
    print("\n" + "=" * 60)
    print("诊断结果")
    print("=" * 60)
    
    passed = sum(1 for _, r in results if r)
    total = len(results)
    
    for name, result in results:
        status = "✅" if result else "❌"
        print(f"{status} {name}")
    
    print(f"\n总计：{passed}/{total} 通过")
    
    if passed == total:
        print("\n🎉 所有检查通过！系统运行正常。")
        return 0
    else:
        print("\n⚠️ 部分检查失败，请查看上面的错误信息。")
        return 1

if __name__ == '__main__':
    sys.exit(main())
```

运行诊断：

```bash
cd ~/.openclaw/workspace
python3 scripts/diagnose.py
```

## 二、常见问题速查

### 2.1 文档创建问题

| 问题 | 可能原因 | 解决方案 |
|------|----------|----------|
| 模板文件不存在 | 文件未创建/路径错误 | 检查 `docs/templates/` 目录 |
| 变量替换失败 | 变量名不匹配 | 检查 `{{variable}}` 格式 |
| 图片占位符未清理 | 正则表达式不匹配 | 检查 `image[[...]]` 格式 |
| 文件夹 token 错误 | token 无效/权限不足 | 从飞书 URL 获取正确 token |

### 2.2 Agent 通信问题

| 问题 | 可能原因 | 解决方案 |
|------|----------|----------|
| Agent 离线 | 网关未启动/Agent 未运行 | `openclaw gateway start` |
| Agent 繁忙 | 并发任务过多 | 等待或增加 `max_concurrent` |
| 发送超时 | 网络问题/Agent 无响应 | 增加 timeout 或检查网络 |
| 死信队列增长 | 持续发送失败 | 检查 Agent 状态和网络 |

### 2.3 任务队列问题

| 问题 | 可能原因 | 解决方案 |
|------|----------|----------|
| 任务超时 | 任务复杂/资源不足 | 增加 timeout 或优化任务 |
| 队列阻塞 | 并发数达到上限 | 增加 `max_concurrent` |
| 状态文件损坏 | 异常退出/磁盘问题 | 删除 `.task-queue-state.json` |
| 监控告警频繁 | 任务运行过长 | 优化任务或调整阈值 |

### 2.4 测试问题

| 问题 | 可能原因 | 解决方案 |
|------|----------|----------|
| pytest 未安装 | 依赖缺失 | `pip install pytest pytest-mock pytest-asyncio` |
| 测试失败 | 环境配置错误 | 检查测试夹具和 mock |
| 覆盖率低 | 测试用例不足 | 添加更多测试场景 |
| 性能不达标 | 系统资源不足 | 检查 CPU/内存/磁盘 IO |

### 3.4 日志分析示例

#### 死信消息分析

```json
{
  "target": "clawbuilder",
  "message": "实现登录功能",
  "error_type": "AgentOfflineError",
  "error_message": "Agent clawbuilder 离线",
  "attempts": 3,
  "failed_at": "2026-03-23T18:00:00"
}
```

**分析**:
- 错误类型：AgentOfflineError
- 重试次数：3 次
- 解决方案：检查 Agent 状态，启动网关

#### 任务队列状态分析

```json
{
  "running": 5,
  "max_concurrent": 5,
  "queue_size": 10,
  "active_tasks": [
    {
      "agent_id": "clawbuilder",
      "task": "实现登录功能",
      "status": "running",
      "started_at": "2026-03-23T17:50:00"
    }
  ]
}
```

**分析**:
- 并发数已达上限 (5/5)
- 队列中有 10 个等待任务
- 解决方案：增加 `max_concurrent` 或优化任务执行时间

## 四、性能问题

### 4.1 性能基准

| 指标 | 目标 | 单位 |
|------|------|------|
| 文档创建 | < 1.0 | 秒/文档 |
| 变量替换 | < 10 | 毫秒/次 |
| 图片清理 | < 5 | 毫秒/次 |
| Agent 通信 | < 5.0 | 秒/次 (含重试) |
| 任务调度 | < 1.0 | 秒/任务 |

### 4.2 性能测试

```bash
# 运行性能测试
pytest tests/test_integration.py::TestPerformance -v -s
```

### 4.3 性能优化建议

#### 文档创建优化

```python
# ✅ 推荐：批量创建
docs = []
for i in range(10):
    doc = creator.create_from_template(
        template='task-template',
        variables={'task_id': f'TASK-{i:03d}'}
    )
    docs.append(doc)

# ❌ 避免：频繁 IO
for i in range(10):
    doc = creator.create_from_template(...)
    # 每次都写入磁盘
```

#### Agent 通信优化

```python
# ✅ 推荐：减少重试次数
await send_with_retry(target, message, max_retries=2)

# ❌ 避免：过多重试
await send_with_retry(target, message, max_retries=10)  # 延迟过长
```

#### 任务队列优化

```python
# ✅ 推荐：合理设置并发
manager = TaskQueueManager(max_concurrent=5)

# ❌ 避免：并发过高
manager = TaskQueueManager(max_concurrent=50)  # 资源竞争
```

### 4.4 性能调优参数

| 参数 | 默认值 | 推荐范围 | 说明 |
|------|--------|----------|------|
| `max_concurrent` | 5 | 3-10 | 最大并发任务数 |
| `base_delay` | 5.0 | 2.0-10.0 | 重试基础延迟 |
| `max_delay` | 60.0 | 30.0-120.0 | 重试最大延迟 |
| `timeout` | 300 | 60-600 | 任务超时时间 |
| `alert_threshold` | 600 | 300-1200 | 告警阈值 (秒) |

## 五、飞书集成问题

### 5.1 API 认证问题

#### 错误：Invalid app_id or app_secret

**原因**:
- 凭证配置错误
- 应用未启用
- 权限不足

**解决方案**:
```bash
# 1. 检查 .env 文件
cat ~/.openclaw/workspace/.env

# 2. 验证凭证
curl -X POST https://open.feishu.cn/open-apis/auth/v3/tenant_access_token/internal \
  -H "Content-Type: application/json" \
  -d '{"app_id":"xxx","app_secret":"xxx"}'

# 3. 检查飞书开发者后台
# 确保应用已启用且有正确权限
```

### 5.2 文档创建问题

#### 错误：folder not found

**原因**:
- folder_token 错误
- 文件夹已删除
- 权限不足

**解决方案**:
```bash
# 1. 从飞书 URL 获取正确 token
# URL 格式：https://xxx.feishu.cn/drive/folder/FRMvfUXEZlvHO1djkYDcwT1PnWh
# token: FRMvfUXEZlvHO1djkYDcwT1PnWh

# 2. 检查文件夹权限
# 确保应用有写入权限

# 3. 更新配置
# 编辑 scripts/feishu_doc_creator.py 中的 TEMPLATES 配置
```

#### 错误：permission denied

**原因**:
- 应用权限不足
- 文件夹权限限制

**解决方案**:
```bash
# 1. 检查飞书应用权限
# 开发者后台 -> 应用 -> 权限管理

# 2. 添加必要权限
# - 云文档读写
# - 云空间读写

# 3. 重新授权应用
```

### 5.3 网络问题

#### 错误：Connection timeout

**原因**:
- 网络连接问题
- 防火墙限制
- API 限流

**解决方案**:
```bash
# 1. 检查网络连接
curl -I https://open.feishu.cn

# 2. 检查防火墙
# 确保 443 端口开放

# 3. 检查 API 限流
# 飞书免费层：60 次/分钟
# 如超限，等待或升级套餐
```
