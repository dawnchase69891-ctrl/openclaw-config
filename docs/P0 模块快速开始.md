### 3. 使用示例

#### 场景 1: 发送消息给 Agent (带重试)

```python
from scripts.collaboration_utils import send_with_retry, AgentOfflineError

try:
    success = await send_with_retry(
        target='clawbuilder',
        message='请实现用户登录功能',
        max_retries=3
    )
    print(f"发送成功：{success}")
except AgentOfflineError as e:
    print(f"Agent 离线：{e}")
except Exception as e:
    print(f"发送失败：{e}")
```

#### 场景 2: 提交任务到队列 (带并发控制)

```python
from scripts.task_queue_manager import TaskQueueManager

manager = TaskQueueManager(max_concurrent=5)

try:
    result = await manager.spawn_with_control(
        agent_id='clawbuilder',
        task='实现用户登录功能',
        priority=5,
        timeout=300
    )
    print(f"任务完成：{result}")
except TimeoutError:
    print("任务超时")
except Exception as e:
    print(f"任务失败：{e}")
```

#### 场景 3: 配置同步 (从飞书)

```bash
# 设置环境变量
export FEISHU_APP_ID=cli_xxx
export FEISHU_APP_SECRET=xxx
export FEISHU_BITABLE_TOKEN=bascnxxx

# 执行同步
python3 scripts/sync_feishu_to_openclaw.py
```

## 命令行工具

### collaboration_utils.py

```bash
# 检查 Agent 状态
python3 scripts/collaboration_utils.py check main

# 发送消息
python3 scripts/collaboration_utils.py send clawbuilder "请实现功能"

# 查看死信队列
python3 scripts/collaboration_utils.py list-dead-letters

python3 scripts/collaboration_utils.py retry /path/to/dead_letter.json
```

```bash

python3 scripts/task_queue_manager.py status

python3 scripts/task_queue_manager.py submit clawbuilder "实现功能"

python3 scripts/task_queue_manager.py monitor
```

## 故障排查

### 问题：Agent 离线

```bash
# 1. 检查 Agent 状态
python3 scripts/collaboration_utils.py check main

# 2. 查看死信队列
python3 scripts/collaboration_utils.py list-dead-letters

# 3. 查看 OpenClaw 会话
openclaw sessions list
```

### 问题：任务超时

```bash
# 1. 查看队列状态
python3 scripts/task_queue_manager.py status

# 2. 增加超时时间
await manager.spawn_with_control(..., timeout=600)
```

### 问题：配置同步失败

```bash
# 1. 检查环境变量
echo $FEISHU_APP_ID
echo $FEISHU_APP_SECRET

# 2. 查看备份
ls -la ~/.openclaw/workspace/.backup/config/

# 3. 手动恢复
python3 -c "from scripts.sync_feishu_to_openclaw import ConfigSyncService; ConfigSyncService().restore_from_backup()"
```
