## 使用指南

### 快速开始

#### 1. 安装依赖

```bash
cd ~/.openclaw/workspace
pip install -r requirements-p1.txt
```

#### 2. 配置环境变量

```bash
# 飞书 API (配置同步)
export FEISHU_APP_ID=cli_xxx
export FEISHU_APP_SECRET=xxx
export FEISHU_BITABLE_TOKEN=xxx

# 飞书 Webhook (通知)
export FEISHU_WEBHOOK_URL="https://open.feishu.cn/open-apis/bot/v2/hook/xxx"
```

#### 3. 配置 GitHub Secrets

在 GitHub 仓库设置中添加:
- `FEISHU_APP_ID`
- `FEISHU_APP_SECRET`
- `FEISHU_BITABLE_TOKEN`

#### 4. 测试验证

```bash
# 配置验证
python3 scripts/validate_config.py

# 飞书通知测试
python3 scripts/feishu_notifier.py test

# 运行测试
pytest tests/ -v
```

### 典型使用场景

#### 场景 1: 手动同步配置

```bash
python3 scripts/sync_feishu_to_openclaw.py
```

#### 场景 2: 发送告警

```python
from feishu_notifier import send_alert
send_alert("内存不足", "系统内存使用率超过 90%")
```

#### 场景 3: 任务完成通知

```python
from feishu_notifier import send_task_complete
send_task_complete(
    task_id="TASK-001",
    assignee="ClawBuilder",
    task_description="实现用户登录功能"
)
```
