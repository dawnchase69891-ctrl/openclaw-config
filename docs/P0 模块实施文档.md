## 一、sessions_send 可靠性增强

### 1.1 功能特性

- ✅ Agent 状态检查 (`is_agent_online`, `get_agent_status`)
- ✅ 指数退避重试 (`send_with_retry`)
- ✅ 死信队列 (`enqueue_dead_letter`)
- ✅ 异常分类 (`AgentOfflineError`, `AgentBusyError`, `SessionSendError`)

### 1.2 安装使用

```python
# 导入模块
from scripts.collaboration_utils import (
    send_with_retry,
    check_agent_status,
    get_dead_letters,
    AgentOfflineError,
    AgentBusyError
)

# 示例 1: 发送消息 (带重试)
try:
    success = await send_with_retry(
        target='clawbuilder',
        message='请实现用户登录功能',
        max_retries=3
    )
except AgentOfflineError as e:
    print(f"Agent 离线：{e}")
except AgentBusyError as e:
    print(f"Agent 繁忙：{e}")

# 示例 2: 检查 Agent 状态
status = await check_agent_status('main')
print(f"Agent 状态：{status.value}")

# 示例 3: 查看死信队列
dead_letters = await get_dead_letters()
for msg in dead_letters:
    print(f"失败消息：{msg.target} - {msg.error_type}")
```

### 1.3 命令行工具

```bash
# 检查 Agent 状态
python3 scripts/collaboration_utils.py check main

# 发送消息
python3 scripts/collaboration_utils.py send clawbuilder "请实现登录功能"

# 查看死信队列
python3 scripts/collaboration_utils.py list-dead-letters

# 重试死信消息
python3 scripts/collaboration_utils.py retry /path/to/dead_letter.json
```

### 1.4 重试机制

| 尝试次数 | 延迟时间 | 说明 |
|---------|---------|------|
| 第 1 次 | 5 秒 | 基础延迟 |
| 第 2 次 | 10 秒 | 指数增长 |
| 第 3 次 | 20 秒 | 指数增长 |
| 最大 | 60 秒 | 上限 |

公式：`delay = min(5 * 2^attempt + jitter, 60)`

### 1.5 死信队列

**存储位置**: `~/.openclaw/workspace/.dead-letter/`

**文件格式**:
```json
{
  "target": "clawbuilder",
  "message": "请实现登录功能",
  "error_type": "AgentOfflineError",
  "error_message": "Agent clawbuilder 离线",
  "attempts": 3,
  "failed_at": "2026-03-23T10:00:00",
  "metadata": {"task_id": "TASK-001"}
}
```

## 二、任务队列优化

### 2.1 功能特性

- ✅ 并发控制 (`max_concurrent=5`)
- ✅ 超时控制 (`default_timeout=300s`)
- ✅ 优先级队列 (1-10, 越高越优先)
- ✅ 状态持久化 (`.task-queue-state.json`)
- ✅ 监控告警 (10 分钟长任务告警)

### 2.2 安装使用

```python
from scripts.task_queue_manager import TaskQueueManager

# 创建管理器
manager = TaskQueueManager(max_concurrent=5)

# 提交任务
result = await manager.spawn_with_control(
    agent_id='clawbuilder',
    task='实现用户登录功能',
    priority=5,        # 优先级 1-10
    timeout=300        # 超时时间 (秒)
)

# 查看队列状态
status = await manager.get_queue_status()
print(json.dumps(status, indent=2))
```

### 2.3 命令行工具

```bash
# 查看队列状态
python3 scripts/task_queue_manager.py status

# 提交任务
python3 scripts/task_queue_manager.py submit clawbuilder "实现登录功能"

# 启动监控
python3 scripts/task_queue_manager.py monitor
```

### 2.4 并发控制

```
最大并发数：5
当前运行：3/5
等待队列：2

⏳ 并发数已达上限 (5/5)，任务进入队列
```

### 2.5 超时告警

| 超时类型 | 阈值 | 动作 |
|---------|------|------|
| 任务超时 | 300 秒 (默认) | 抛出 TimeoutError，记录失败状态 |
| 长任务告警 | 600 秒 | 发送告警通知 (防抖 10 分钟) |

### 2.6 状态持久化

**存储位置**: `~/.openclaw/workspace/.task-queue-state.json`

**文件格式**:
```json
{
  "running": 2,
  "active_tasks": [
    {
      "agent_id": "clawbuilder",
      "task": "实现登录功能",
      "priority": 5,
      "timeout": 300,
      "created_at": "2026-03-23T10:00:00",
      "started_at": "2026-03-23T10:00:05",
      "status": "running"
    }
  ],
  "last_updated": "2026-03-23T10:00:05"
}
```

## 五、集成示例

### 5.1 在 auto_task_scheduler.py 中使用

```python
from scripts.collaboration_utils import send_with_retry, AgentOfflineError
from scripts.task_queue_manager import TaskQueueManager

# 初始化队列管理器
queue_manager = TaskQueueManager(max_concurrent=5)

async def assign_task(agent_id: str, task: str, priority: int = 5):
    """分配任务给 Agent"""
    try:
        # 使用队列管理
        result = await queue_manager.spawn_with_control(
            agent_id=agent_id,
            task=task,
            priority=priority,
            timeout=300
        )
        return True
    except AgentOfflineError:
        # Agent 离线，记录日志并告警
        log_error(f"Agent {agent_id} 离线")
        send_alert(f"Agent 离线：{agent_id}")
        return False
    except asyncio.TimeoutError:
        # 任务超时
        log_error(f"任务超时：{agent_id}")
        return False
```

### 5.2 在 CI/CD 中使用

```yaml
# .github/workflows/sync-config.yml
```

## 六、故障排查

### 6.1 常见问题

**问题 1**: `AgentOfflineError: Agent xxx 离线`

**解决**:
1. 检查 Agent 是否正在运行：`openclaw sessions list`
2. 检查死信队列：`python3 scripts/collaboration_utils.py list-dead-letters`
3. 重试死信消息：`python3 scripts/collaboration_utils.py retry <filepath>`

**问题 2**: `任务超时 (300 秒)`

**解决**:
1. 增加超时时间：`timeout=600`
2. 检查任务复杂度，考虑拆分
3. 查看监控日志：`python3 scripts/task_queue_manager.py status`

**问题 3**: `配置验证失败：'systemPrompt' is a required property`

**解决**:
1. 检查飞书多维表格配置
2. 确保所有必填字段都有值
3. 查看备份配置：`ls -la ~/.openclaw/workspace/.backup/config/`

### 6.2 日志位置

| 日志类型 | 位置 |
|---------|------|
| 死信队列 | `~/.openclaw/workspace/.dead-letter/` |
| 状态文件 | `~/.openclaw/workspace/.task-queue-state.json` |
| 备份配置 | `~/.openclaw/workspace/.backup/config/` |
