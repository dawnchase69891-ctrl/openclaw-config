## 📁 技术规格

### 技术栈
- **Python 3.7+**
- **quantstats** - 投资组合分析
- **yfinance** - 数据获取
- **pandas** - 数据处理
- **matplotlib** - 图表绘制

### 文件结构
```
~/.openclaw/workspace/
├── scripts/
│   └── quantstats_report.py      # 报告生成脚本
├── reports/
│   └── quantstats_report.html    # HTML 报告
├── config.json                    # 持仓配置
└── FINANCIAL_AGENT.md            # Agent 配置
```

### API 接口
```python
# 示例代码
import quantstats as qs
import yfinance as yf

# 下载数据
data = yf.download('000703.SZ', start='2025-01-01')

# 计算收益
returns = data['Close'].pct_change()

# 生成报告
qs.reports.html(returns, output='report.html')
```
