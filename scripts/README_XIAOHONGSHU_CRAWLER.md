## 📊 数据采集流程

```
1. 启动浏览器 → 2. 检查登录状态 → 3. 搜索关键词
                                          ↓
5. 保存数据 ← 4. 抓取笔记详情 ← 滚动加载更多
```

### 详细步骤

1. **浏览器初始化**
   - 启动 Chromium 浏览器
   - 使用用户数据目录（复用登录态）
   - 注入反检测脚本

2. **登录检测**
   - 自动检测是否已登录
   - 未登录时提供 60 秒手动登录时间

3. **搜索关键词**
   - 访问搜索页面
   - 滚动加载更多笔记
   - 提取笔记 URL

4. **抓取详情**
   - 访问每篇笔记
   - 提取标题、作者、互动数据
   - 提取完整文案和标签

5. **数据存储**
   - 导出为 JSON 文件
   - 可导入飞书多维表格

## 🐛 常见问题

### Q1: 浏览器无法启动

**解决**:
```bash
# 重新安装 Playwright
pip uninstall playwright
pip install playwright
playwright install chromium
```

### Q2: 提示未登录

**解决**:
- 脚本会等待 60 秒供手动登录
- 登录成功后会自动继续
- 下次运行会复用登录态

### Q3: 抓取数据为空

**可能原因**:
- 小红书页面结构变化
- 账号被限制
- 网络问题

**解决**:
- 检查日志文件 `xiaohongshu_crawler.log`
- 手动访问小红书验证账号状态
- 调整选择器（需更新脚本）

### Q4: 飞书导入失败

**检查**:
- App Token 是否正确
- Table ID 是否正确
- 字段名称是否匹配
